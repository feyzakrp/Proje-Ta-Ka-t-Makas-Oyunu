# Pygame Projesi

Bu proje, Pygame kullanılarak yapılmış basit bir oyun içerir. Oyun, Python ve Pygame kütüphanesi kullanılarak geliştirilmiştir ve oyun sırasında arka planda müzik çalar.

## Özellikler
